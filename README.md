# Mini LMS Backend

A self-owned demo backend project showcasing clean architecture, JWT authentication,
and role-based access control.

## Tech Stack
- Node.js
- Express
- JWT
- Jest

## Features
- Authentication with JWT
- Role-based access control
- Modular architecture
- Basic test coverage

## Setup
1. Clone repository
2. npm install
3. Copy .env.example to .env
4. npm run dev

## Notes
This repository is created solely for evaluation and demonstration purposes.
